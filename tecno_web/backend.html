Programación del lado del Servidor

La programación del lado del servidor es una tecnología que reside en el procesamiento de una solicitud de usuario mediante la interpretación de un script en el servidor web con la finalidad de generar páginas HTML de forma dinámica como respuesta a esa solicitud
